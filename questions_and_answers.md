###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: `{}`

<i>Here, 'greetign' is declared globally. And assigns a value `{}`. So when prints the value of 'greetign', it prints `{}`</i>

</p>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: `"12"`

<i>When the 'sum' function calls, there was 2 values as arguments. First one is number type and the second one is string type. we know, the sum of a number and a string will always be string. so the function concatenates as string. And the answer is `"12"`.</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: `Hi there, undefined`

<i>'sayHi' function has a parameter named 'name'. When function calls, if there is no argument passed, the value of the parameter 'name' will be `undefined`. so the answer is `Hi there, undefined`.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: `3`

<i>here, 0 is a falsy value. So in the first iteration, count remains 0. In the last three iteration, the count increased by 1. When the forEach loop completed the execution, the value of count will be `3`. Thats why, the code prints `3`.</i>

</p>
</details>
